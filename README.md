**Морской Бой - Сетевая Игра**

**Описание проекта**

*Это сетевая реализация классической игры "Морской бой" для двух игроков. Проект состоит из двух компонентов:*//
-Сервер - управляет игровыми сессиями, сопоставлением игроков и логикой игры
-Клиент - позволяет игрокам подключаться к серверу и участвовать в сражениях

*Особенности*
Сервер:
-Многопоточная архитектура для обработки множества подключений
-Автоматический подбор игроков (матчмейкинг)
-Автоматическая расстановка кораблей
-Управление игровыми сессиями
-Статистика сервера в реальном времени
-Обработка отключений игроков
Клиент:
-Простой и интуитивно понятный интерфейс
-Валидация ввода пользователя
-Отображение игрового поля и вида противника
-Автоматическое восстановление при ошибках сети

*Технические требования*
ОС: Windows (используются WinSock API)
Компилятор: Поддерживающий C++11 или выше
Библиотеки: Winsock2

*Компиляция*
Компиляция сервера:
bash
g++ -o NavalBattle_server.exe NavalBattle_server.cpp -lws2_32 -std=c++11
Компиляция клиента:
g++ -o NavalBattle_client.exe NavalBattle_client.cpp -lws2_32 -std=c++11

*Запуск*
1. Запуск сервера:
NavalBattle_server.exe
Сервер запросит порт (по умолчанию 12345) и начнет прослушивать подключения.
2. Запуск клиентов:
NavalBattle_client.exe

Каждый клиент запросит:
IP-адрес сервера (по умолчанию 127.0.0.1)
Порт сервера (по умолчанию 12345)

*Игровой процесс:*
Клиенты подключаются к серверу
Сервер автоматически сопоставляет игроков в пары
Корабли расставляются автоматически
Игроки по очереди делают ходы, вводя координаты (x y)
Игра продолжается до потопления всех кораблей одного из игроков

*Управление сервером*
Во время работы сервер поддерживает команды администратора:
/stats - показать статистику сервера
/stop - остановить сервер
/help - показать список команд

*Правила игры*
Состав флота:
1 корабль на 4 клетки
2 корабля на 3 клетки
3 корабля на 2 клетки
4 корабля на 1 клетку

Обозначения на поле:
"." - неизвестная клетка
"S" - корабль (видно только на своем поле)
"X" - попадание
"O" - промах
"#" - потопленный корабль

Формат хода:
Игрок вводит координаты в формате:
x y
где x и y - числа от 0 до 9

*Особенности реализации*
Безопасность:
Валидация всех пользовательских вводов
Обработка ошибок сети
Защита от некорректных данных
